# Generate Git Commands

## Purpose
Produce standard Git command suggestions for a specific task or workflow.

## Prompt
```text
Provide standard Git commands for the requested task.

For each command, include:
- Command
- What it does
- When to use it
- A simple example

Rules:
- Keep the commands practical and commonly used
- Prefer safe commands before destructive ones
- Use placeholders for branch names, remotes, paths, or tags
- Keep explanations concise
- Group related commands if helpful
```

## Notes
- Replace placeholders with real values before running commands
