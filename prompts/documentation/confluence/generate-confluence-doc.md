# Generate Confluence Documentation

## Purpose
Turn raw technical or business input into a reusable Confluence-style document.

## Prompt
```text
Generate a Confluence-ready document from the provided input.

Include sections such as:
- Title
- Summary
- Background or Context
- Details
- Steps or Process
- Risks or Notes
- Action Items

Rules:
- Keep the structure clean and easy to paste into Confluence
- Use clear headings and concise wording
- Adapt section names to the topic if needed
- Use placeholders for missing information
- Keep the output professional and tool-agnostic
```

## Notes
- Tailor the section list if the document type is known
